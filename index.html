Project 1: Slime Madness

2D Platformer, written in C

Slime Madness is a 2d platformer, with unique character manipulation mechanics. The controlled player character can be split at will, and the player can select portions of itself to control. This project was developed entirely in C. We used a now open source graphics library, CProcessing (https://github.com/DigiPen-Faculty/CProcessing). Audio implementation was completed using FMOD. My contribution to this project was gameplay development. I implemented the player controller functionality.

My Contributions:

  • User input
  • Player Selection
  • Split player memory management

Project 5: Apocalypse Resort

Project Description

I joined Apocalypse Resort for one semester. During my time on this project, I implemented a telemetry logging solution to provide hard data for the psychology researcher we were building this project for.
